# Install

## Before installing

Read boot.txt to understand the discipline this system expects.

## What gets installed (T0 core)

```
~/.subtract/
  handler.sh      # command_not_found hook
  subtract        # main script
  lookup.tsv      # your vocabulary (fork layer)
  hooks/          # shell integration
```

This is T0 only. Higher tiers via `subtract upgrade` after install.

## Requirements

- bash 4.0+
- curl

## Inspect first

```
curl -sL subtract.ing/install.sh
```

Read it. It's 50 lines. Then run it.

## Install

```
curl -sL subtract.ing/install.sh | bash
```

## Verify

Open new terminal, then:

```
subtract --version
```

## Uninstall

```
rm -rf ~/.subtract
# Remove source line from ~/.bashrc or ~/.zshrc
```

## Platforms

- Linux (x86_64, arm64)
- macOS (arm64, x86_64)
- WSL2

## No account required

Runs entirely local. No API key. No telemetry.
