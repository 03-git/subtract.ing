# Substrate

The entire subtract stack requires zero dependencies that aren't already on a Unix system or don't ship as a single file.

No interpreter. No package manager. No runtime. No SDK.

## Tiers

See tiers.txt for architecture and upgrade menu.

## Fork layer (~/.subtract/)

trigger.conf     — PTT, toggle, hotword
transcribe.conf  — dash→-, dot→.
lookup.tsv       — personal vocabulary
governance.conf  — confirm patterns

## Voice as tier

Voice isn't an accessibility "feature" gated behind a fork. It's a tier like any other, visible in the same menu as local inference. The person who needs voice input sees it alongside the person who needs offline reasoning. Same menu, same dignity, same opt-in.

The fork layer handles how you use voice (trigger, transcription rules), not whether you can access it.
