# Tiers

## Architecture

T0      bash function     already there
T0.5    man               already there
T1      embed binary      ships
T2      llama binary      ships
T-Voice whisper binary    ships
T-Ear   piper binary      ships
T4      curl              already there

Every tier is either "already there" or "single binary + model."

## Upgrade menu

subtract upgrade
[1] Local inference (905MB) — T1+T2, reasoning
[2] Voice input (150MB) — T-Voice, whisper.cpp
[3] Voice output (50MB) — T-Ear, piper-tts
[4] Voice bundle (200MB) — T-Voice + T-Ear
[5] Cloud escalation — T4, API key
[6] Cancel

All tiers visible. Opt-in. No tier hidden behind another.
