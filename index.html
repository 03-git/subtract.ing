When you're ready for automation, subtract is a shell that does what you mean.

$ ./install.sh

Try: type subtract

Run 'subtract upgrade' for optional tiers.

$ type subtract
subtract is /home/user/.subtract/subtract

$ subtract upgrade

Optional tiers:
[1] Local inference (905MB) — T1+T2, reasoning
[2] Voice input (150MB) — T-Voice, whisper.cpp
[3] Voice output (50MB) — T-Ear, piper-tts
[4] Voice bundle (200MB) — T-Voice + T-Ear
[5] Cloud escalation — T4, requires API key
[6] Cancel

Choice: 5

Cloud escalation (T4) uses curl to call inference APIs.
